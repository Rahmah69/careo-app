# careo-app

1. Clone and Install

# Clone the repo
git clone https://github.com/Rahmah69/careo-app

# Navigate to clonned folder and Install dependencies
cd careo-app & yarn install

# Install Pods
cd ios & pod install

2. Build and Run the app in your iOS simulator

# build
cd ../ & yarn build:ios

# run
yarn start

# in another terminal
yarn run:ios
